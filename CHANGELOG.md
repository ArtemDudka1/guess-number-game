## v0.14 — Achievement System

### Новое
- Добавлена полноценная система достижений.
- Создана база из 56 достижений в 7 категориях:
  - Игры.
  - Победы.
  - Серии побед.
  - Ставки.
  - Прогресс.
  - Особые достижения.
  - Скрытые достижения.
- Добавлены пять уровней редкости: обычные, редкие, эпические, легендарные и мифические достижения.
- Добавлены награды XP за получение достижений.
- Добавлена цепная проверка условий: награда одного достижения может открыть следующее достижение или новый уровень.
- Добавлена поддержка достижений за количество игр, побед, серии, ставки, уровни и суммарный XP.
- Добавлены особые достижения за уникальные игровые сценарии.
- Добавлены пять скрытых достижений, которые не отображаются до получения.
- Добавлены последовательные условия с собственным прогрессом.
- Добавлена ретроактивная проверка накопительной статистики после загрузки старого сохранения.

### Экран достижений
- Создан отдельный экран достижений.
- Добавлено отображение достижений по категориям.
- Добавлены карточки с названием, описанием, наградой, редкостью и текущим прогрессом.
- Добавлена цветовая индикация редкости.
- Добавлено отображение завершённых достижений и заполненной полосы прогресса.
- Добавлена поддержка прокрутки длинного списка категорий.
- Добавлен общий счётчик полученных достижений в header экрана.
- До получения первой скрытой ачивки общий счётчик показывает прогресс из 51 обычного достижения.
- После получения первой скрытой ачивки общий счётчик показывает прогресс из всех 56 достижений.
- Добавлен отдельный счётчик скрытых достижений из 5, который появляется после получения первой скрытой ачивки.
- В каждый контейнер категории добавлен собственный счётчик выполненных достижений.
- Увеличена верхняя область контейнеров категорий для размещения счётчиков без перекрытия карточек.

### Уведомления о достижениях
- Добавлен Popup получения достижений.
- Добавлена поддержка одновременного получения нескольких достижений.
- Добавлены сортировка наград по типу и редкости и отдельное оформление скрытых достижений.
- Добавлена прокрутка Popup при получении большого количества достижений.
- Добавлено отображение суммарной награды XP.
- Добавлено уведомление о получении нового титула.
- Добавлена кнопка перехода из Popup на экран достижений.

### Профиль и титулы
- Добавлен счётчик достижений в профиль игрока.
- Добавлен процент выполнения всех достижений.
- Добавлена система титулов за достижение определённых уровней.
- Добавлен автоматический выбор самого высокого доступного титула.
- Добавлен финальный титул «Абсолютный коллекционер» за получение 56 из 56 достижений.
- Расширена статистика профиля по играм, победам, сериям и ставкам.

### Developer-панель
- Добавлена developer-панель для тестирования системы достижений.
- Добавлена выдача и откат отдельных достижений по ID.
- Добавлена выдача и откат всех достижений.
- Добавлены быстрые команды начисления XP и игровых очков.
- Добавлена очередь developer-команд во время активной партии.
- Изменения применяются только после подтверждения кнопкой «Готово».
- Добавлена отмена подготовленных изменений без изменения прогресса.
- Добавлена проверка цепных XP- и level-достижений после developer-команд.
- Явно откатываемые достижения защищены от повторной выдачи в том же пакете изменений.
- Developer-панель постоянно доступна через главное меню и сочетание `Cmd + 1` на macOS.

### Сохранение и совместимость
- Добавлено сохранение состояния всех 56 достижений.
- Добавлено сохранение вспомогательной статистики особых и скрытых достижений.
- Добавлена безопасная инициализация достижений при первом запуске без файла сохранения.
- Добавлена совместимость со старыми сохранениями без полей системы достижений.
- Добавлена проверка типов загружаемых данных.
- Повреждённое сохранение теперь резервируется рядом с игрой с расширением `.corrupt`.
- Добавлена атомарная запись прогресса через временный файл.
- Путь к сохранению больше не зависит от рабочей директории запуска игры.
- Добавлена возможность задавать отдельный путь сохранения через переменную окружения для тестирования.

### Исправлено
- Исправлено падение при первом запуске без `save_data.json`.
- Исправлено падение при загрузке повреждённого JSON.
- Исправлен конфликт имени состояния экрана и базы достижений.
- Исправлены достижения «Несокрушимый» и «Абсолютный риск», которые ранее не могли открыться.
- Исправлена проверка достижений с отдельным полем прогресса.
- Исправлено условие достижения за ставку всем балансом от 1000 монет: победа больше не требуется.
- Исправлена граница достижения «Большой куш»: требуется выиграть именно более 5000 очков.
- Исправлена проверка XP- и level-достижений после ручного начисления XP.
- Исправлена цепная проверка после ручной выдачи награды через developer-панель.
- Исправлена повторная выдача явно откатываемых достижений внутри одного developer-пакета.
- Исправлена обработка скрытой категории при отсутствии полученных скрытых достижений.
- Устранено дублированное определение функции переноса текста.
- Удалены неиспользуемые старые debug-функции и пустые функции отдельного сохранения достижений.
