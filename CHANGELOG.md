# Changelog

Все значимые изменения проекта.

## v0.1 - Начало проекта

### Добавлено
- Создан проект игры "Угадай число".
- Базовая игровая механика.
- Генерация случайного числа.
- Проверка ответа игрока.
- Победа и поражение.

## v0.2 - Сложность игры

### Добавлено
- Лёгкая сложность.

## v0.9 - Главное меню

### Добавлено

- Полностью переработано главное меню.
- Добавлена система всплывающих окон (попапов).
- Выбор сложности перенесён в попап.
- Добавлены разделы:
  - Профиль
  - Достижения
  - Настройки
- Добавлены информационные попапы для разделов в разработке.
- Улучшена структура навигации по игре.

### Изменено

- Статистика убрана с главного экрана.
- Главное меню стало основой для будущих разделов игры.

## v0.11 - Text Input System

### Добавлено
- Создан универсальный компонент TextInput.
- Добавлена отрисовка текстового поля.
- Добавлена система активации поля по клику.
- Добавлен ввод текста с клавиатуры.
- Добавлена фильтрация разрешённых символов.
- Добавлено ограничение максимальной длины текста.
- Добавлена поддержка удаления символов (Backspace).
- Добавлена поддержка удержания клавиш.
- Добавлена поддержка снятия фокуса клавишей Escape.
- Добавлен мигающий курсор.
- Улучшено отображение placeholder во время ввода.
